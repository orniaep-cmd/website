Development

MCP

Minecraft Coder Pack (now known as Mod Coder Pack) is used to decompile, change, and recompile Minecraft's client and server. It supports alpha v1.1.2_01 to release 1.12.

MCP-Reborn

MCP-Reborn is an MCP (Mod Coder Pack) for Minecraft for making modded clients and researching its code. It supports release 1.13 and onwards.

Weave Loom

Weave Loom, used for creating mods. Unlike Forge and Fabric, Weave isn't for vanilla Minecraft, but rather for Lunar Client.

Forge MDK

Forge MDK, used for creating mods. One of the oldest, most supported, and most popular mod loaders for Minecraft.

Fabric API

Fabric API, used for creating mods. Unlike Forge, it offers support for Snapshot versions of the game.

Optifine SRC

Optifine SRC for adding Optifine to your MCP clients. If the website is down, please use the backup download.

Programs

EzPing

EzPing is a software that modifies your Registry in an attempt to give you better ping and FPS. Cracked by Sapphire Industries.

Clumsy

Clumsy is a program that simulates broken network conditions. You can ping spoof and fakelag without using a hacked client!

Java Decompilers

Java decompilers can turn a compiled .jar or .class into readable code, mostly useful for checking for viruses.

HostsMod

Blocks connections to viruses in some Minecraft cheats (such as Kant's viruses) and IP loggers.
